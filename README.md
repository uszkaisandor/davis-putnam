# davis-putnam  

Java implementation of the original Davis-Putnam algorithm (1960).  

### Prerequisites

JDK8  
IntelliJ IDEA or Android Studio

### Input format

The Main class reads a CNF from standard input.  
Format examples:  
(x)  
((r)∧(¬r))  
((r)∧(¬r∨¬s)∧(s)∧(p∨s))
